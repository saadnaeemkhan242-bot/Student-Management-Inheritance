# Student Management System - Single Inheritance

This Java program demonstrates the concept of **single inheritance**.

## Classes
1. **Person**: Stores personal information (name, ID, age)
2. **Student**: Inherits from `Person` and adds academic info (marks, grade)

## Features
- Demonstrates single inheritance in Java
- Uses constructors for initialization
- Simple and beginner-friendly
